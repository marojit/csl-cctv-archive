# csl-cctv-archive

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中存放的页面内容来源于不同来源，经过整理后以静态 HTML 形式保存，方便查阅与分发。本项目不针对任何单个域名或具体网站，仅作为页面存档的集合。

## 目录说明

```
csl-cctv-archive/
├── pages/               # 存放归档的 HTML 页面
├── assets/              # 页面引用的静态资源（如图片、样式、脚本）
└── README.md            # 本文件
```

- **pages/**：每个子目录或文件对应一个独立的 HTML 页面归档。
- **assets/**：页面所需的公共资源，按需组织。

## 页面归档说明

- 每个归档页面均为独立完整的 HTML 文件，可离线打开浏览。
- 页面内容保持原始结构，仅对无法加载的外部资源做本地化处理（如有）。
- 归档时间戳记录在页面元数据或文件命名中，方便追溯。
- 不保证所有页面长期可用，但会尽力维护存档的完整性。

## 维护说明

- 本仓库不定期更新，添加新的归档页面或修复已有页面中的资源链接。
- 提交新页面时，请尽量保持原有格式，避免引入无关内容。
- 如发现页面无法正常显示，欢迎提交 Issue 或 Pull Request。
- 本仓库仅作存档用途，不提供内容解读或附加说明。

## 许可

本仓库中的页面内容版权归各自原始作者所有。归档行为仅用于保存与查阅，不涉及任何商业用途。如需转载或引用，请遵守原始内容的许可条款。
